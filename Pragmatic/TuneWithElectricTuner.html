How to

Tune a Guitar with an Electronic Tuner

Step 1: Tune the sixth (thickest) string

👌🗘☌

  1. Confirm the tuner is set to the E string.
  2. Pluck the sixth string repeatedily while observing the electronic tuner.
    Ensure only the sixth string is plucked.
  3. IF...THEN...
    Tuner indicates string is flat ...Tighten
    Tuner indicates string is sharp ...Loosen
    Tuner indicates string is in tune ...Go to next step (tune the next string)
